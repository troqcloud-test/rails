# Deploy to TroqCloud

This is a sample [Ruby on Rails](https://rubyonrails.org/) app set up to work with on TroqCloud.

The sample app is deployed at https://rails.troqcloud.app.

## Deploy to TroqCloud

Fork the repo and use the button below to deploy this app one click.

<a href="https://troqcloud.com/deploy" referrerpolicy="no-referrer-when-downgrade" rel="nofollow">
  <img src="https://troqcloud.com/images/deploy-to-troqcloud-button.svg" alt="Deploy to TroqCloud" />
</a>

To deploy manually, see the guide at https://troqcloud.com/docs
